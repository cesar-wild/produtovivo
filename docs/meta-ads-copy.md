## Configuração Técnica

### Eventos do Pixel a configurar (via Meta Events Manager)

| Evento | Onde | Código |
|--------|------|--------|
| `PageView` | Todas as páginas | Automático via base code |
| `Lead` | POST /leads (quiz submit) | `fbq('track', 'Lead')` |
| `InitiateCheckout` | Click em "Comprar agora" | `fbq('track', 'InitiateCheckout')` |
| `Purchase` | /obrigado (após pagamento) | `fbq('track', 'Purchase', {value: 37, currency: 'BRL'})` |

### Custom Audiences a criar

1. Visitantes do site (todos) — últimos 30 dias
2. Visitantes do /quiz — últimos 14 dias
3. Visitantes do /#comprar — últimos 7 dias (mais quentes)
4. Lista de e-mails coletados (upload CSV)

### Lookalikes a criar (depois de ter 100+ compradores)

1. 1% LLA baseado em compradores (Purchase event)
2. 1% LLA baseado em lista de e-mails
3. 2% LLA para escalar
