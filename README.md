# ProdutoVivo

Premium educational guide for Brazilian infoproduct creators. Transform PDFs into interactive AI-powered apps.

## Structure

- `/src` — Express.js backend (PostgreSQL, Stripe, Resend)
- `/public` — Landing page, blog, static assets
- `/product` — Downloadable guide files
- `/docs` — Strategy, brand, marketing docs
- `/assets/ads` — Meta Ads creatives
- `/agents` — Agent instruction files

## Stack

- **Backend:** Express.js + PostgreSQL + Stripe
- **Email:** Resend (@produtovivo.com)
- **Ads:** Meta Ads (Pixel 4520253334926563)
- **Domain:** produtovivo.com
- **Dev:** Container at :8082 on Hetzner
