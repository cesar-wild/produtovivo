Política de Privacidade

Última atualização: abril de 2026

Esta política descreve como o ProdutoVivo (produtovivo.com) coleta, usa e protege suas informações pessoais.

1. Informações coletadas

Coletamos apenas o necessário para processar sua compra e entregar o produto:

  • Nome e e-mail — fornecidos no checkout (via Hotmart ou Kiwify)
  • Dados de pagamento — processados diretamente pela plataforma de pagamento; não armazenamos dados de cartão
  • Dados de uso do site — via Meta Pixel (comportamento de navegação para fins de publicidade)

2. Como usamos suas informações

  • Entrega do produto digital comprado
  • Envio de e-mails transacionais (confirmação de compra, acesso ao produto)
  • Melhoria das campanhas publicitárias via Meta Pixel
  • Suporte ao cliente

3. Compartilhamento de dados

Não vendemos nem compartilhamos seus dados pessoais com terceiros, exceto:

  • Plataformas de pagamento (Hotmart, Kiwify) — para processamento da compra
  • Meta (Facebook/Instagram) — dados de uso do site para exibição de anúncios relevantes
  • Resend — para envio de e-mails transacionais

4. Meta Pixel

Este site usa o Meta Pixel (ID: 4520253334926563) para medir a eficácia de anúncios e exibir publicidade relevante. Você pode optar por não participar em facebook.com/adpreferences.

5. Seus direitos (LGPD)

Nos termos da Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Acessar, corrigir ou excluir seus dados pessoais
  • Revogar o consentimento para uso dos seus dados
  • Solicitar portabilidade dos dados

Para exercer esses direitos, entre em contato: contato@produtovivo.com

6. Segurança

Adotamos medidas técnicas e organizacionais razoáveis para proteger seus dados contra acesso não autorizado, perda ou divulgação indevida.

7. Contato

Dúvidas sobre esta política? contato@produtovivo.com
