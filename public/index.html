Prompt C-01

Transformar capítulo de PDF em chatbot

Você é um assistente especialista em [TEMA DO SEU PRODUTO].

Abaixo está o conteúdo do capítulo [N]:
"""
[COLE O TEXTO AQUI]
"""

Sua tarefa:
1. Responder perguntas com base neste conteúdo
2. Usar linguagem simples e direta
3. Quando não souber, dizer "isso está fora do escopo deste capítulo"
4. Terminar cada resposta com uma pergunta de acompanhamento
Prompt Q-03

Gerar quiz de diagnóstico a partir do conteúdo

Com base no conteúdo abaixo, crie um quiz de diagnóstico com 5 perguntas:

"""
[COLE O TRECHO DO SEU MATERIAL]
"""

Formato para cada pergunta:
- Enunciado claro e direto
- 4 alternativas (A, B, C, D)
- Gabarito + explicação de 1 linha
- Nível: iniciante / intermediário / avançado

Objetivo: ajudar o leitor a identificar onde está e o que precisa focar.
Prompt V-07

Escrever página de vendas para produto interativo

Escreva uma página de vendas para o produto abaixo.

Produto: [NOME]
Formato: [ebook + chatbot / quiz / glossário]
Público: [DESCRIÇÃO]
Preço: R$[VALOR]
Principal resultado: [O QUE O COMPRADOR CONSEGUE]

Estrutura obrigatória:
1. Headline com dor + resultado
2. Agitação do problema (3 parágrafos)
3. Apresentação da solução
4. O que está incluído (lista)
5. Para quem é / não é
6. CTA com garantia de 7 dias
