A Epidemia de Ansiedade Infantil

O Brasil registra crescimento significativo de diagnósticos de ansiedade e TDAH em crianças. Pais, professores e psicólogos buscam ferramentas acessíveis para ajudar crianças a desenvolver regulação emocional. Mindfulness adaptado para a infância é uma dessas ferramentas, com evidências científicas sólidas.

O Que Incluir no Produto

Exercícios de respiração para crianças (adaptados por faixa etária), práticas de atenção plena de 5 minutos, histórias meditativas guiadas, como ensinar crianças a nomear emoções, mindfulness na hora de dormir (para pais e filhos), e como incorporar mindfulness na rotina familiar são os conteúdos principais.

Dois Produtos em Um

Um produto de mindfulness para crianças é, na prática, um produto para pais e professores — são eles os compradores e os facilitadores das práticas. Crie materiais tanto para as crianças (as práticas em si) quanto para os adultos (como mediar as práticas).

Perguntas Frequentes

Preciso ser psicólogo para criar um produto de mindfulness para crianças?

Formação em psicologia infantil, pedagogia ou instrução de mindfulness (MBSR adaptado para crianças) é muito importante. Para práticas clínicas terapêuticas, sim, é necessário ser profissional de saúde mental.

Para qual faixa etária deve ser o produto?

3-6 anos (práticas muito simples, com pais), 7-10 anos (práticas guiadas autônomas) e 11-14 anos (mindfulness adolescente) são as três faixas principais. Produtos por faixa têm mais conversão.
