Perguntas Frequentes

Culinária vegana é difícil de aprender?

Para quem já cozinha, não. O maior desafio é substituir ingredientes com sucesso. Cursos e guias que ensinam as substituições certas têm valor imenso para iniciantes.

Nutricionista precisa assinar e-book de receitas veganas?

Para receitas gerais, não. Para planos alimentares com fins terapêuticos, sim. E-books de receitas são conteúdo educativo livre.

Como precificar produtos de culinária vegana?

E-books de receitas: R$27-R$67. Cursos completos: R$197-R$397. Planos semanais personalizados: R$67-R$147.
