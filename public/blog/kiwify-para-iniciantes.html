Kiwify para Iniciantes: Guia Completo 2026

A Kiwify cresceu muito nos últimos dois anos e hoje compete de igual para igual com o Hotmart no Brasil. Para infoprodutores com audiência própria, a Kiwify tem taxa menor e interface mais moderna. Este guia mostra o passo a passo completo — da criação da conta até o primeiro saque.

Criando sua conta na Kiwify (passo a passo)

  1. Acesse kiwify.com.br → "Criar conta grátis"
  2. Cadastre com email + senha + CPF/CNPJ
  3. Confirme o email (link de verificação)
  4. Complete o perfil: nome completo, data de nascimento, endereço
  5. Adicione dados bancários para receber saques (PIX ou conta bancária)

Criando seu primeiro produto na Kiwify

Produtos do tipo "Ebook / PDF"

  1. Painel → "Produtos" → "Criar produto" → "Conteúdo digital"
  2. Preencha: nome do produto, descrição, imagem de capa (1200x630px)
  3. Faça upload do arquivo (PDF, ZIP, etc.)
  4. Configure o preço: valor único, parcelado (até 12x) ou PIX com desconto
  5. Configure página de obrigado (URL do seu site ou usar a padrão da Kiwify)
  6. Publique — fica disponível imediatamente

Produtos do tipo "Curso online"

  1. Painel → "Produtos" → "Criar produto" → "Curso"
  2. Crie módulos e aulas (drag-and-drop)
  3. Faça upload dos vídeos diretamente (Kiwify usa Bunny CDN — velocidade alta)
  4. Configure acesso: vitalício ou com prazo (ex: 1 ano)
  5. Configure certificado (opcional — a Kiwify gera automaticamente)
  6. Publique

Configurando o checkout da Kiwify

O checkout da Kiwify é um dos mais rápidos do mercado. Configure estes elementos:

  • Imagem do produto: aparece no checkout — use imagem que transmita valor
  • Depoimentos: adicione 2-3 depoimentos diretamente no checkout (aumenta conversão em 15-25%)
  • Order bump: produto complementar oferecido com um clique antes de finalizar
  • Garantia: destaque a garantia de 7 dias — obrigatória por lei para produtos digitais (CDC)
  • Formas de pagamento: PIX (desconto opcional), cartão de crédito, boleto
  • PIX automático: ative o desconto de 5-10% no PIX — aumenta conversões em mobile

Order bump: como configurar na Kiwify

O order bump é o produto complementar oferecido no próprio checkout. É adicionado com um clique — sem precisar de nova compra. Taxa de aceitação média: 15-35% dos compradores.

  1. Crie o produto complementar separado (ex: template, bônus premium)
  2. No produto principal → "Checkout" → "Order Bump"
  3. Selecione o produto complementar e configure o preço do bump
  4. Escreva a headline do bump: "Adicione [produto] por apenas R$[X]" (máx. 2 linhas)

Sistema de afiliados na Kiwify

Para ativar afiliados:

  1. No produto → "Afiliados" → "Ativar programa de afiliados"
  2. Configure a comissão (% ou valor fixo por venda)
  3. Convide afiliados pelo email — eles recebem link único de afiliado
  4. Acompanhe as vendas por afiliado em "Relatórios → Afiliados"

Comissão recomendada: 30-50% para afiliados — infoprodutos têm margem alta e afiliados motivados por comissão maior se dedicam mais à divulgação.

Quanto tempo para receber na Kiwify

Fluxo de pagamento padrão:

  • Venda aprovada → D+14 (14 dias após a aprovação)
  • Saque mínimo: R$50
  • Formas de saque: PIX (instantâneo após liberação) ou TED (D+1)
  • Período de proteção contra chargeback: os 14 dias servem para garantir que o cliente não cancele

FAQ — Kiwify para iniciantes

Kiwify ou Hotmart: qual escolher para iniciantes?

Se você já tem audiência própria: Kiwify (menor taxa, interface mais ágil, aprovação instantânea). Se quer se beneficiar do marketplace de afiliados da plataforma: Hotmart (maior base de afiliados já cadastrados procurando produtos). Para iniciantes sem audiência: Hotmart. Com audiência própria: Kiwify.

Quanto a Kiwify cobra por venda?

6,99% + R$0,99 por venda aprovada. Para um produto de R$97: taxa de R$7,78. Sem mensalidade — você paga apenas sobre vendas. Saque mínimo de R$50, disponível 14 dias após a venda.

Kiwify aceita afiliados?

Sim, mas não tem marketplace público como o Hotmart — você convida seus próprios afiliados. Para quem tem afiliados identificados (ex-alunos, parceiros do nicho), funciona muito bem. Para quem quer encontrar afiliados novos, o Hotmart tem vantagem.
