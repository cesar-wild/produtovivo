Perguntas Frequentes

Preciso ser técnico em TI ou elétrica para criar este produto?

Conhecimento técnico prático é importante. Para automações simples (sem obras), qualquer entusiasta com experiência real pode criar um produto de qualidade.

Qual o melhor ecossistema para ensinar — Google, Amazon ou Apple?

Google Home e Amazon Alexa têm maior penetração no Brasil. Ensinar um dos dois (ou ambos) é o melhor caminho. HomeKit é mais nichado para usuários Apple.
