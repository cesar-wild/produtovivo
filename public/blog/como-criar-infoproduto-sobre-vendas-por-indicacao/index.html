Perguntas Frequentes

Qual é o melhor incentivo para programa de indicação?

Depende do perfil do cliente. Para consumidores: desconto ou crédito na próxima compra. Para empresas B2B: porcentagem em dinheiro ou upgrade de serviço. Reconhecimento público funciona bem para profissionais liberais. Teste com seu público específico.

Quando pedir indicação para o cliente?

Após uma entrega excepcional, quando o cliente expressou satisfação espontânea, logo após a resolução de um problema de forma surpreendente ou no momento de renovação do contrato. Nunca logo após uma reclamação.
