Perguntas Frequentes

Qual a idade ideal para começar a programar?

Pensamento computacional pode começar aos 5-6 anos com atividades unplugged. Programação visual (Scratch) funciona bem a partir dos 7-8 anos.

Scratch ou Python para crianças?

Scratch para até 10 anos (interface visual sem código), Python a partir dos 11 anos. A transição natural do Scratch para Python funciona muito bem.
