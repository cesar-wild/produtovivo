Perguntas Frequentes

Preciso de registro na Vigilância Sanitária para vender comida?

Depende do tipo e escala. Produção doméstica para venda local tem regras mais flexíveis — mas formalize o quanto antes para crescer sem problemas.

Como precificar pratos e sobremesas para venda?

Some ingredientes + embalagem + tempo de produção (hora/valor) + custos fixos proporcionais + margem de 50-100%. A maioria subestima o tempo de produção.
